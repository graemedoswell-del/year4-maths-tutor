
Year 4 Maths Adventure Ultimate

Included files:
- app.py
- curriculum.py
- question_bank.py
- requirements.txt
- progress.json
- templates/index.html
- templates/parent.html
- templates/test.html

Deploy on Render:
Build command:
pip install -r requirements.txt

Start command:
python app.py

Optional:
Add OPENAI_API_KEY in Render environment variables to enable AI explanations for wrong answers.
